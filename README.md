# 📚 Kilde Assistent

Din personlige akademiske kilde- og citatdatabase — som app på telefon og computer.

## Sådan kommer du i gang (5 minutter)

### Trin 1 — Opret et nyt repository på GitHub
1. Gå til **github.com** og log ind
2. Tryk på det grønne **"New"** knap (øverst til venstre)
3. Navngiv det: `kilde-assistent`
4. Sæt det til **Public**
5. Tryk **"Create repository"**

### Trin 2 — Upload filerne
1. På din nye repository-side, tryk **"uploading an existing file"**
2. Træk alle disse filer ind:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Tryk **"Commit changes"**

### Trin 3 — Slå GitHub Pages til
1. Gå til **Settings** (tandhjulet øverst i dit repository)
2. Scroll ned til **"Pages"** i menuen til venstre
3. Under "Source" vælg **"Deploy from a branch"**
4. Vælg **"main"** og **"/ (root)"**
5. Tryk **Save**

### Trin 4 — Din app er live! 🎉
Efter 1-2 minutter er din app tilgængelig på:
`https://DITBRUGERNAVN.github.io/kilde-assistent/`

## Installer som app på telefonen

### iPhone (Safari):
1. Åbn app-linket i **Safari**
2. Tryk på **Del-ikonet** (firkant med pil op)
3. Vælg **"Føj til hjemmeskærm"**
4. Tryk **"Tilføj"**

### Android (Chrome):
1. Åbn app-linket i **Chrome**
2. Tryk på de **tre prikker** øverst til højre
3. Vælg **"Føj til startskærm"**
4. Tryk **"Tilføj"**
